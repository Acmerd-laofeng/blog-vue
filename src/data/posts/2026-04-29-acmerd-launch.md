# Acmerd 企业信息收录平台上线

🎉 **Acmerd** 正式上线了！

这是一个专注于企业信息收录的平台，旨在帮助求职者更好地了解目标企业。

## 平台功能

- 🏢 **企业收录** - 收录企业所在城市、作息制度、薪资范围等信息
- 🔍 **搜索筛选** - 按城市、行业、作息等条件筛选企业
- 📝 **文章系统** - 分享职场经验和求职技巧
- 🔐 **后台管理** - 管理员可方便地增删改查企业信息

## 收录标准信息

每家企业收录以下信息：

| 字段 | 说明 |
|------|------|
| 企业名称 | 企业全称 |
| 所在城市 | 省份 + 城市 |
| 所属行业 | 如：互联网、通信、金融等 |
| 作息时间 | 双休 / 单休 / 大小周 |
| 薪资范围 | 如：15K-30K |
| 综合评分 | 0-5 分 |
| 企业简介 | 简要介绍 |
